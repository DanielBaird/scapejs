Class: ScapeScene

ScapeScene

new ScapeScene(field, dom, options)

Represents a rendering of a landscape / moonscape / whatever

Parameters:
Name Type Description
field ScapeField

the field being rendered
dom string

DOM element the scape should be rendered into.
options object

collection of options. All are optional.

Properties
Name Type Default Description
lights Array.<String> 'sun','sky'

array of strings naming lights to include in this scene. Choose from:

string light type
topleft a light from above the camera's left shoulder
ambient a dim ambient light
sun a directional light that orbits the scene once per day
sky a directional light that shines from above the scene
currentDate Date | "now" 'now'

The time and date inside the scape. The string "now" means set currentDate to the present.
timeRatio number 1

The rate time should pass in the scape, relative to normal. 0.1 means ten times slower. 60 means one minute real time = one hour scape time.
dateUpdate ScapeScene~dateChange

callback for when the scene time changes (which is a lot).

Methods

addBlocks()

add blocks from the attached ScapeField into the scene.

You will probably only need to call this once.

addHelperCube(x, y, z, color)

add a cube at position x, y, z to confirm where that is, exactly. Great for trying to work out if your scape is being rendered where you think it should be rendered.

Parameters:
Name Type Argument Default Description
x Number | Vector3

X coordinate, or a THREE.Vector3 containing x, y and z coords
y Number <optional>

Y coordinate
z Number <optional>

Z coordinate
color Color | String | Integer '#cccccc'

Color of cube. Can be a THREE.Color, a color-parseable string like '#3366cc', or a number like 0x3366cc.

addHelperShapes()

add helper cubes at some of the corners of your scape, so you can see where they are in space.
