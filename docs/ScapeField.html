Class: ScapeField

ScapeField

new ScapeField(options)

The container for all information about an area.

Parameters:
Name Type Description
options object

Various options for the ScapeField being created.

option default value description
minX 0 smallest X for this field
maxX 100 largest X for this field
blocksX 10 number of blocks to divide the X axis into
minY 0 smallest Y for this field
maxY 100 largest Y for this field
blocksY 10 number of blocks to divide the Y axis into
minZ 0 smallest Z (vertical dimension) for this field
maxZ 40 largest Z for this field
blocksZ 80 number of blocks to divide the Z axis into
blockGap 0.04 gap to leave between blocks along the X and Y axes

addGroundHeight(x, y, z)

Add a claim that the ground height is z at point x,y. If you call this, remember to eventually call calcGroundHeights after so ground heights get extrapolated across the entire Field.

X coordinate of this ground height record

the height of the ground at position x,y

addGroundHeights(heightList, replace)

Add a list of claims of the ground height at various points. Unlike addGroundHeight, this method will re-extrapolate ground heights across the Field (so you don't need to call calcGroundHeights yourself).

Parameters:
Name Type Description
heightList Array

A list of objects. Each element must have x, y, and z properties.
replace Boolean

If a truthy value is supplied, this method will discard existing height claims before adding these ones. If false or unsupplied, these new claims will be added to the existing ones.

addGroundStack()

Add a ground stack at x,y, starting at height z. The stack is an array of two-element arrays with a Material and a depth number, like this: [ [Material.leafLitter, 0.3], [Material.dirt, 3.5], [Material.stone, 4] ] That puts a leaflitter layer 0.3 units deep on a 3.5-unit deep dirt layer, which is on a stone layer. If the final layer's depth is zero, that layer is assumed to go all the way to minZ. If you call this, remember to calcGround() after.

addGroundStacks(replace)

Add additional ground stacks to the field's ground stacks. The groundList is an array of data objects. Each object needs x, y and z properties, and a 'stack' property, each matching the corresponding arg to addGroundStack.

Parameters:
Name Type Description
replace boolean

if replace is truthy, discard existing ground points first.

addItems(itemList, replace)

Add a list of items to the scape at various points. Unlike addItem, this method will re-position items across the Field (so you don't need to call calcItems yourself).

Parameters:
Name Type Description
itemList Array

A list of items. Each element must have x, y, and item properties.
replace Boolean

If a truthy value is supplied, this method will discard existing height claims before adding these ones. If false or unsupplied, these new claims will be added to the existing ones.

calcGroundHeights()

(re)calculate the ground height. You need to call this if you add ground height claims one at a time using addGroundHeight.

calcGroundStacks()

(re)calculate the ground stacks. You need to call this if you add ground stacks one at a time using addGroundStack.
